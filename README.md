## :cd: Installation

```sh
git clone git@github.com:filipe1309/dotr.git
```

```sh
cd dotr
```

## :runner: Running

```sh
./vendor/bin/sail up
```

> Access http://localhost

## :white_check_mark: Tests

After up the container:

```sh
docker-compose exec -t dotr_laravel ./vendor/bin/phpunit
```
